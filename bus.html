Real Time Bus Tracking

In this exercise the goal is to work with a mapping library to pull in data asynchronously and display map elements.

The project file structure:

  • mapanimation.js: This file contains the javascript code that receives the data, parse it and use it to animate the moving of the bus
  • index.html: This is the HTML file that displays the movement of the bus.
  • style.css: This contains the style classes declared for various elements used in index file.
